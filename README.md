# Staring

![Staring icon](https://github.com/zhenyi/Staring/raw/gh-pages/icon.png)

Staring is a simple menu bar app that’ll keep your Mac from sleep.
<br>
Actually, I did this mostly because I wanted to have a look of disapproval on my menu bar.

![Staring screenshot](https://github.com/zhenyi/Staring/raw/gh-pages/screenshot.png)

## Installation

1. Download the [zip file](https://github.com/zhenyi/Staring/raw/gh-pages/Staring.zip).
2. Extract it and copy Staring.app into your Application folder.
